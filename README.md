# RobloxMod Chip System Documentation

This folder contains documentation for the in-game programmable Chip system.

- `docs/README.md`: Overview and quick start.
- `docs/API.md`: Runtime API exposed to user code.
- `docs/LANGUAGE.md`: Supported Lua subset and syntax rules.
- `docs/EDITOR.md`: In-game editor usage.
- `docs/DEPLOYMENT.md`: Tool placement, lifecycle, and runtime behavior.
